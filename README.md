# AICCRA_CP_Tool
Modified version of the IGP climate tool to inform a GCF proposal for Tanzania

If you're having issues running the markdown, try directly specifying the location of the project directory.

Note the revision and improvement this tool is supported by the AICCRA project. An earlier version of the tool was supported by EiA https://github.com/EiA2030/eia_IGP_climate_tool.
